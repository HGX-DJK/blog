# MVT-Process
Mapbox矢量瓦片解析、调试、处理工具，消息在控制台中打印 [demo](https://bingqixuan.github.io/magic/index.html#/tools/mvt)

-----------------------------------
## 安装
```bash
   npm install
   
   npm install --save-dev parcel
```

## 运行
```bash
    npx parcel index.html
```

## 功能
- [x] 支持单个pbf文件图层信息解析
- [x] 支持单个pbf文件绘制
